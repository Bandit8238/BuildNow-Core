# BuildNow-Core # 🎮 BuildNow Core

Advanced game launcher ecosystem for BuildNow.gg.  
Mods, plugins, and version management — all in one place.

---

## ⚡ Features

- 🔄 Auto-update system
- 🧩 Mod loader (JS injection)
- 🔌 Plugin system
- 📦 Version manager
- 🔐 Secure Electron setup

---

## 🚀 Installation

```bash
git clone https://github.com/yourname/buildnow-nexus.git
cd buildnow-nexus
npm install
npm start
```

---

## 🧱 Structure

```
core/        → backend systems
renderer/    → UI
mods/        → game mods
plugins/     → launcher plugins
versions/    → game builds
```
